Returns the exponent power of the number

- Create a base`a` and Exponent `b`, and then use power(a,b) which returns the exponent power of the number 'a'.
- Consider a as the base and b as the exponent

```
const power = (a, b) =>
    return Math.pow(a,b));
  );
```

```js
power(7, 2); // 49
```
